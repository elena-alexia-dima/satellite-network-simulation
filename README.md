A simulation of a satellite communication network implemented in C,
using binary tree structures and Min-Heap algorithms for efficient
node management and routing.
The simulation focuses on efficient data routing, encoding and decoding
of transmitted messages, and computing distances between nodes in the network.

Features:
- Binary tree–based network topology representation
- Min-Heap structure for priority-based node management
- Custom encoding and decoding routines for message transmission
- Distance computation algorithms between network nodes
- Optimized performance for large-scale network simulation

Technologies:
- Language: C
- Data Structures: Binary Trees, Min-Heap
- Tools: GCC, Make, Valgrind
